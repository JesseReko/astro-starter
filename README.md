# Astro Business Website Starter

Reusable Astro starter for small business websites.

## Edit client info
Update:

- `src/config/site.js`
- `src/data/services.js`
- `src/data/valuePoints.js`

## Run locally

```bash
npm install
npm run dev
```

## Pages

- `/`
- `/services`
- `/about`
- `/contact`

## Structure

- `src/components/ui` = header/footer/global UI
- `src/components/sections` = reusable page sections
- `src/config` = site-wide business info
- `src/data` = editable content lists
